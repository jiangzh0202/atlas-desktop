📡 通道配置教程

将 AI 员工接入飞书、企业微信、邮件，让你的团队直接对话 AI。

1
创建飞书应用
打开 飞书开放平台，点击「创建企业自建应用」。
记下 App ID（类似 cli_a...）和 App Secret。
2
添加机器人能力
进入应用详情 → 左侧「添加应用能力」→ 添加「机器人」。
配置机器人的名称、头像、描述。保存后，上方导航栏才会出现「事件订阅」等菜单。
3
配置权限（批量导入）
左侧「权限管理」→ 右上角「批量添加」。

📋 点击展开查看权限 JSON（共 27 项） 

        

        
        ✅ 已复制
        

        
          展开确认格式无误后点击复制 → 粘贴到飞书弹窗。也可直接复制上方 JSON 上传。
        
      

    


    

      
4

      

        配置事件订阅

        左侧「事件订阅」→ 请求网址填：
        https://atlas.traceclaw.cn/api/feishu/callback
        添加事件：接收消息 → 搜索 im.message.receive_v1 并添加。

        保存后，系统会发验证请求（Atlas 自动应答）。

        获取页面上的 Verification Token，填入 Atlas 控制台中。
      

    


    

      
5

      

        发布应用

        左侧「版本管理」→ 创建版本 → 提交发布。

        ⚠️ 必须发布后，回调地址才会在正式环境生效！发布前，机器人只响应应用创建者。
      

    


    

      
6

      

        填入 Atlas 控制台并测试

        返回 Atlas 控制台 → 设置 → 通道 → 飞书。

        填入以下三个字段：
        

          
App IDcli_a...（步骤1获取）

          
App Secret步骤1获取

          
Verification Token步骤4获取

        

        点击「🔗 测试连接」，看到绿色提示即表示配置成功。
      

    

  


  
  

    

      
1

      

        创建企业微信应用

        打开 企业微信管理后台 →「应用管理」→ 创建自建应用。
      

    

    

      
2

      

        获取认证信息

        在应用详情页获取 Corp ID（企业ID）、Agent ID 和 Secret。
      

    

    

      
3

      

        配置接收消息

        「接收消息」→ 设置回调 URL，填入：
        https://atlas.traceclaw.cn/api/wecom/callback
        随机生成 Token 和 EncodingAESKey（43位），保存并填入 Atlas 控制台。
      

    

    

      
4

      

        设置可信域名

        「网页授权及JS-SDK」→ 设置可信域名为 atlas.traceclaw.cn。
      

    

    

      
5

      

        填入 Atlas 控制台并测试

        返回 Atlas 控制台 → 设置 → 通道 → 企业微信。
        

          
Corp IDww...（企业ID）

          
Agent ID1000001

          
App Secret步骤2获取

          
Token步骤3随机生成

          
EncodingAESKey步骤3随机生成（43位）

        

        点击「🔗 测试连接」。
      

    

  


  
  

    

      
1

      

        准备 SMTP 信息

        Atlas 通过 SMTP 发送邮件。你需要一个支持 SMTP 的邮箱账号（如企业邮箱、QQ邮箱、163邮箱等）。
      

    

    

      
2

      

        获取授权码（如需要）

        QQ邮箱：设置 → 账户 → POP3/SMTP服务 → 开启并获取授权码。

        163邮箱：设置 → POP3/SMTP/IMAP → 开启并获取授权码。
      

    

    

      
3

      

        填入 Atlas 控制台并测试

        返回 Atlas 控制台 → 设置 → 通道 → 邮件。
        

          
SMTP 服务器smtp.qq.com 或 smtp.163.com 等

          
SMTP 端口465（SSL）或 587（STARTTLS）

          
邮箱账号你的完整邮箱地址

          
邮箱密码/授权码邮箱密码或 SMTP 授权码

        

        点击「🔗 测试连接」。
      

    

  



  
  

    

      MCP (Model Context Protocol) 是 AI 与外部系统通信的标准协议。

      Atlas 通过 MCP 让你的数字员工直接操作 ERP、CRM、邮件、物流等系统——不是看数据，是替你干活。
    


    

      
1

      

        准备 MCP 服务器

        MCP 服务器可以是你自己部署的，也可以使用开源方案。


        已适配的开源 MCP：
        

          
🏭 ERPnpm: atlas-mcp-erp用友/金蝶库存、采购、财务

          
👥 CRMnpm: mcp-hubspot客户画像、商机、跟进

          
📧 邮箱npm: mcp-server-email邮件搜索、发送、附件

          
⚙️ MaaSnpm: atlas-mcp-maas工单、质检、排产

          
🚚 物流npm: atlas-mcp-logistics运单查询、轨迹追踪

        

      

    


    

      
2

      

        部署 MCP 服务器

        Atlas 支持 HTTP 传输模式的 MCP 服务器。你需要：
        


        方式一：使用 Atlas 官方包

        npx atlas-mcp-erp --port 3456 --api-key YOUR_KEY
        

        方式二：自建 MCP 服务器

        任何符合 MCP 规范 的 HTTP 服务器，暴露 / 端点接收 JSON-RPC 请求即可。
        


        ⚠️ 建议使用 HTTPS 和 API Key 保护你的 MCP 端点。API Key 会通过 Authorization: Bearer 头发送。
      

    


    

      
3

      

        在 Atlas 中接入

        打开 Atlas 控制台 → 设置 → 🔌 MCP → 选择要接入的系统 → 点击「接入」。


        弹出窗口中填写：
        

          
MCP 服务器地址https://your-server.com:3456

          
API 密钥部署时设置的密钥（可选但推荐）

        

        先点「🔍 测试连接」验证，再点「💾 保存并接入」。
      

    


    

      
4

      

        验证接入成功

        保存后，已连接的 MCP 服务器会出现在列表中。数字员工在对话中会自动调用对应的工具。


        示例对话：

        
          你：「查一下康明斯 ISF3.8 发动机的库存和最新采购价」

          AI：「正在查询 ERP 系统… ISF3.8 当前库存 126 台，最新采购价 ¥3,850/台（供应商：东风康明斯，更新于 6月18日）。」
        
      

    


    

      
5

      

        MCP 与通道的区别

        

          
通道AI 通过飞书/企微/邮件与你沟通

          
MCPAI 连接到你的业务系统（ERP/CRM 等）执行操作

        

        通道处理「人→AI→人」的对话，MCP 处理「AI→系统→AI」的数据操作。两者结合，数字员工才能真正替你干活。

🤖 AI 模型全局配置

    
所有 AI 调用（报价分析、客户开发、智能回复）共用此配置。在管理后台修改后实时生效，无需重启。


    

      
1

      

        配置入口

        管理后台 → 🤖 模型配置 → 🌐 全局 AI 配置（绿色边框区域）
      

    


    

      
2

      

        配置项说明


        

          
服务商deepseekAI 服务提供商（支持任意 OpenAI 兼容 API）

          
默认模型deepseek-v4-pro推理能力最强的旗舰模型

          
API Basehttps://api.deepseek.com/v1API 端点地址，换服务商时修改此项

          
API Keysk-...***...密钥自动脱敏，输入新 Key 覆盖旧值

        

      

    


    

      
3

      

        System Prompt（系统提示词）

        定义 AI 的角色、语气、知识范围。例如：

        你是擎天 Atlas 的 AI 助手，帮助汽车零部件行业...


        模式参数 — 三种调用模式可独立调参：

        

          
⚡ 快速Temp 0.7 / 300 tokens日常对话、简单查询

          
🔍 深度Temp 0.3 / 2000 tokens复杂报价分析、技术文档

          
💾 存档Temp 0.5 / 1500 tokens知识库训练、数据解析

        

      

    


    

      
4

      

        兜底回复 — AI 异常时的用户提示：

        

          
空回复嗯，收到了。有什么我可以帮您的？

          
解析错误抱歉，我暂时无法处理，请稍后再试。

          
网络异常网络异常，请稍后再试。

          
超时回复超时，请重新发送。

          
系统异常抱歉，出了点问题，请再试一次。

        

      

    


    

      
⚡

      

        即时生效

        配置保存后 下次 AI 调用即生效，无需重启服务。

        配置文件位置：/srv/atlas/data/ai_config.json
      

    

  


  
  

    
🎯 客户开发数字员工

    
AI 驱动的外贸获客引擎 — 整合 7 大工具，自动搜索潜在客户、绘制客户画像、挖掘联系方式。


    

      
🔍

      

        核心能力

        客户开发数字员工内置以下获客技能，自动组合使用：


        

          
ImportGenius海关进口采购记录查真实买家、采购品类、价格区间

          
ImportYeti竞争对手客户分析反向查对手供应了哪些客户

          
Panjiva全球物流追踪跟踪买家物流记录、获取决策人信息

          
RocketReach直接联系方式30秒内获取CEO电话/邮箱

          
Apollo邮箱电话挖掘批量获取客户联系方式

          
SimilarWeb客户来源分析分析客户网站流量来源和渠道

          
Google Trends搜索趋势监测监测目标市场产品搜索热度

        

      

    


    

      
1

      

        使用方法 — 在桌面客户端中直接对话

        打开 Atlas 桌面客户端 → 选择「🔍 客户开发员」→ 输入需求：

        "帮我找美国的康明斯配件进口商，要5家以上，有联系方式的优先"
      

    


    

      
2

      

        画像拼图 — 多维度客户分析

        AI 自动聚合以下维度构建客户画像：

        • 采购记录（ImportGenius/Panjiva）→ 买过什么、买多少、什么价

        • 竞争对手分析（ImportYeti）→ 和谁合作、供应商是谁

        • 联系方式（RocketReach/Apollo）→ 是谁、怎么联系

        • 行为习惯（SimilarWeb/Google Trends）→ 从哪来、搜什么

        
💡 画像越完整，报价命中率越高。建议至少收集 3 个以上维度再出价。

      

    


    

      
3

      

        邮件开发信 — AI 自动撰写

        选定目标客户后，AI 根据客户画像自动生成个性化开发信：

        "给刚才找到的 5 家公司写开发信，英文，突出我们的康明斯配件价格优势"


        开发信会自动适配：

        • 客户行业术语

        • 当地商务习惯

        • 产品优势卖点

        • 报价锚点设计
      

    


    

      
⚙️

      

        配置说明

        • 客户开发员使用全局 AI 配置（见 🤖 AI配置 Tab）

        • 搜索范围：发动机配件、商用车配件、工程机械配件

        • 支持 5 语言自动适配：中文/英文/西班牙语/阿拉伯语/俄语

        • 所有线索自动存入 Atlas 客户库，可在桌面端查看和管理
